---

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Run development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### 3. Build for production

```bash
npm run build
npm run start
```

---

## Deploy to Vercel

1. Push to GitHub
2. Import project at [vercel.com/new](https://vercel.com/new)
3. Vercel auto-detects Next.js — click **Deploy**

Or use Vercel CLI:
```bash
npm i -g vercel
vercel
```

## Project Structure

```
src/
  app/
    globals.css        # Global styles, fonts, animations
    layout.tsx         # Root layout with metadata
    page.tsx           # Main page (imports all components)
  components/
    Navbar.tsx         # Sticky gold/navy navbar
    Hero.tsx           # Hero + project brief card + stats
    Services.tsx       # 4 service cards (Residential, Commercial, Land, Construction)
    About.tsx          # Company history timeline + stats
    WhyChooseUs.tsx    # 6 trust pillars
    Projects.tsx       # Featured projects with filter tabs
    Testimonials.tsx   # Client testimonials
    Contact.tsx        # Contact form → WhatsApp + direct call
    Footer.tsx         # Links, services, contacts
public/
  favicon.ico          # Add your favicon here
  logo.png             # Add company logo here
```
